# 研究简报（中文）

## 1. 目标会议要求

目标会议：**NeurIPS / ICML** —— 顶级机器学习会议，有严格的页数限制、双盲审稿和截止日期驱动的投稿周期。

| 要求 | 详情 |
|---|---|
| 格式 | LaTeX（会议 style 文件，如 `neurips_2025.sty`）；双盲；无作者信息 |
| 页数限制 | 8 页正文 + 不限参考文献/附录（NeurIPS 2025）；9 页（ICML 2025） |
| 摘要 | 单段，约 150-200 词，置于首页顶部 |
| 结构 | 紧凑 IMRaD；相关工作通常压缩至 0.5-1 页；方法部分必须高效地论证设计选择 |
| 图表 | 嵌入文中；矢量图优先；必须灰度可读；图注简洁 |
| 代码/数据 | 强烈建议发布代码；制品评估可选 |
| AI 使用政策 | 需披露 LLM 使用情况；许多会议现要求 AI 使用声明 |
| 补充材料 | 无限制附录，用于证明、消融实验、超参数、扩展结果 |

## 2. 审稿标准

会议审稿人在时间压力下（每篇论文约 30-45 分钟）进行评估。他们关注：

1. **前 2 页的新颖性信号**——如果审稿人读完引言后无法陈述贡献，论文就有风险。研究空白必须清晰，机制对比必须是结构性的，而非修辞性的。
2. **页数限制下的技术严谨性**——正文中的每个论断都必须有正文中可见的证据支持。机制验证是可接受的，但必须展示清晰的因果关系，而非仅仅相关性。
3. **快速可读性**——每节开头、图注和每段首句必须承载论证。以每分钟 200 词浏览的审稿人仍应能跟上逻辑。
4. **诚实的定位**——夸大论断会被扣分。说"我们证明 X 在张力 Y 下是可能的"比"我们解决了 X"得分更高。
5. **可复现性**——环境、超参数、随机种子和代码可获取性很重要。会议审稿人越来越多地检查实验设置是否可复现。

## 3. 已接受论文模式

**模式 1："首页推销"。** 顶级会议论文将完整论证放在第 1 页：首句点明瓶颈，第二段明确空白，第四段引入方法，贡献列表在页末之前出现。没有铺陈。

**模式 2：一个干净的实验故事。** 在页数限制下，成功的论文不会进行 4 个以上不相关的实验。它们运行一个主要实验，将其结构化为叙事（干预前 → 干预 → 干预后 → 反事实），并将消融实验放在附录中。EG-RSA 的 3 个案例研究（有效编辑、死锁、放宽）恰好构成这一叙事弧线。

**模式 3：将相关工作压缩为"定位"而非"综述"。** 会议相关工作部分为 3-4 段，而非 5 个小节。每段涵盖一条研究线索，引用 3-5 篇关键论文，并以与本方法的明确对比结束。该部分以定位段收尾。

## 4. 本文约束

1. **8 页目标。** 当前期刊手稿约 7000 词 / 12 页。必须压缩至约 4500 词 / 8 页。主要压缩目标：合并相关工作子节，精简方法，将消融讨论移至附录。
2. **必须采用首页推销。** 引言必须在页末之前完成完整论证弧线。将贡献列表提高。
3. **不编造基准。** 保持机制验证框架。案例研究即为证据。
4. **无泄漏真实信息论断。** 保持真实评估与搜索决策的严格分离。
5. **审计死锁是标题发现。** 对于会议论文，用最有意思的结果开头。死锁 → 放宽审计的故事比有效编辑案例研究更令人难忘。
6. **附录用于消融实验和扩展轨迹。** 将计划中的消融实验、环境细节和扩展迭代轨迹移至补充附录。
