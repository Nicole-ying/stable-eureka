# EG-RSA v1 Release Design

## Release positioning

`v1release` is the first formal design version of EG-RSA.

Earlier code and experiments are treated as the `v0 baseline`: useful, but not yet a full agentic reward-search system. The v0 system shows that an LLM can tune a human-designed reward schema, but its behavior is still close to a constrained JSON-edit pipeline.

The v1 target is:

**Memory-driven Agentic Reward Search with Tool-augmented Semantic Diagnostics**
